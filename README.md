## Proyecto de Gestión de Inventarios y Ventas

Este proyecto es una aplicación web para la gestión de inventarios, empleados, proveedores, clientes y ventas de productos.

### Tecnologías Utilizadas

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
- JavaScript
